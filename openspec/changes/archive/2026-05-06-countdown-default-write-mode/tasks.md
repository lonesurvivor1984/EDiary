## 1. 修改 diary.c 头部引用

- [x] 1.1 在 `diary.c` 顶部添加 `#include <conio.h>`（提供 `_kbhit()` / `_getch()`）

## 2. 实现倒计时菜单逻辑

- [x] 2.1 在 `main()` 中，将现有的 `printf` 菜单提示替换为带倒计时的循环：初始打印菜单选项，然后进入3次迭代（i = 3, 2, 1）
- [x] 2.2 每次迭代：用 `\r` 刷新同一行，显示当前剩余秒数，例如 `\r请选择 (X 秒后自动写日记): `
- [x] 2.3 每次迭代内调用 `Sleep(1000)`，同时用 `_kbhit()` 轮询（可拆分为每100ms一次，共10次），检测用户按键
- [x] 2.4 检测到按键后用 `_getch()` 读取字符；若为 `'1'` 或 `'2'`，设置 `choice` 变量并 `break` 跳出循环；其他字符忽略
- [x] 2.5 循环结束后若 `choice` 仍未设置，将其默认为 `1`（写日记模式）并打印提示 `\n自动进入写日记模式\n`
